# منصة سيرتك

منصة تعليمية لوزارة التربية والتعليم الفلسطينية، تهدف إلى تقديم محتوى تعليمي تفاعلي للطلاب والمعلمين بشكل ميسّر وحديث.

## الوصف

سيرتك هي منصة تعليمية رقمية تم تطويرها لخدمة المنظومة التعليمية الفلسطينية، وتوفر بيئة تعلم آمنة ومناسبة للجميع.

## كيفية التشغيل

لتشغيل المشروع محلياً، نفّذ الأوامر التالية:

```bash
# تثبيت الاعتماديات
npm install

# تشغيل خادم التطوير
npm run dev
```

ثم افتح المتصفح على العنوان الذي يظهر في الطرفية (عادةً `http://localhost:5173`).

## أوامر أخرى مفيدة

```bash
# بناء النسخة الإنتاجية
npm run build

# معاينة النسخة الإنتاجية
npm run preview
```

## التقنيات المستخدمة

- **React** — مكتبة بناء واجهات المستخدم.
- **Vite** — أداة بناء وتطوير سريعة.
- **Tailwind CSS** — إطار عمل CSS لتصميم الواجهات.
- **React Router** — لإدارة التنقل بين الصفحات.
